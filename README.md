## Features
- Waves met zombie spawners
- Zombie AI (NavMeshAgent)
- Barricades met HP + planken (repair met **F**)
- Points systeem (+10 hit, +50 kill)
- Deuren open met points
- Wallbuys (wapen kopen / ammo kopen)
- 2 weapons, switch met **C**, reload met **R**
- UI: points, prompts, waves + intermission timer

## Tech
- Unity (3D)
- C#
- NavMesh
- TextMeshPro

## Controls
WASD = bewegen  
Shift = sprint  
Space = jump  
F = interact  
LMB = shoot  
R = reload  
C = switch weapon
